# Data-Visualisation-Cruise-Diary
 A data-driven cruise diary using data visualization to visualize data coming in from the sensors of a cruise ship.
 
 
## General description

The aim is to build a narrative tool that tells the story of a sea cruise using the data collected during the cruise. 

## Mockup Dataset

### entries.csv header
appWindDir, appWindSpeed, computedWindDir, computedWindSpeed, distance, elevation, humidity, lat, lon, pressure, shipDirection, shipSpeed, solarRadiation, temperature, time, UVIndex, waterTemperature
